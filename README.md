webRTC
===

## 説明
webRTCを用いたビデオチャットツールです

## 実行手順
```
cd server
node signaling.js
cd ../client
(python3系)
python -m http.server 8000
(python2系)
python -m SimpleHTTPServer 8000
```
ブラウザ(推奨Google chrome)でlocalhost:8000にアクセス
